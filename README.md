angular-oauth-ex
================

Example project using Django REST Framework OAuth2 and an AngularJS client

Install
=======

```bash
pip install requirements.txt
```

Running Django part
===================

```bash
$ python manage.py runserver
```

Running Angular part
====================

```bash
npm install
bower install
grunt server
```

Then go to http://localhost:9000 and login using user:admin password:123456
